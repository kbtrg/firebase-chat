## firebase を用いた複数人リアルタイムチャットアプリ

### 開発

1. yarn
2. firebaseのプロジェクトを作成し、env.local.sampleを元にenv.localを作成
3. firebaseのauthenticationMethodを匿名とメールパスワード方式を有効化
4. yarn dev
5. (PWAを作成する場合は、yarn build & yarn start)
